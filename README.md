Versions & Requirements
-----------------------
1.0.0, PHP >=5.4.1

Installation
------------
The preferred installation method is via composer. You can add the library
as a dependency via:

`$ composer require doctorconceptual/php-instagram-graph-sdk`

Usage
-----
<?php

use Instagram\Instagram;

?>
